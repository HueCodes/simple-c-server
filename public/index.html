Simple C HTTP Server

A multithreaded HTTP server written in C99.

Available Endpoints

GET /
Dynamic route - Welcome page
GET /about
Dynamic route - About page
GET /health
Dynamic route - Health check (JSON)
GET /index.html
Static file - This page

Features

  • POSIX threads for concurrent connections
  • Static file serving from ./public
  • Query string parsing
  • MIME type detection
  • Path traversal protection
